1. 安装

  • 环境

ubuntu 15.10 X64

  • 安装nodejs
sudo apt-get update
sudo apt-get install nodejs 
sudo apt-get install npm
  • 验证
$ node -v
v0.12.2
$ npm -v
2.7.4
  • 安装gitbook
$ > sudo npm install -g gitbook-cli

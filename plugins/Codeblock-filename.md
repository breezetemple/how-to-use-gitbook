# Codeblock-filename

[codeblock-filename](https://plugins.gitbook.com/plugin/codeblock-filename)为代码块添加文件名称

```json
plugins: [ "codeblock-filename" ] 
```
使用示例:  

```bash
''':helloworld
helloworld
'''

'''js:test.js
console.log("test");
'''
```

```:helloworld
helloworld
```

```js:test.js
console.log("test");
```
